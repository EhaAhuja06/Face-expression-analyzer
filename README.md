# Face-expression-analyzer

Realtime emotion detector using DeepFace + OpenCV.

## Files
- `deepface_detect.py` - main local detection script  
- `emotion_app.py` - Streamlit web interface (optional)  
- `haarcascade_frontalface_default.xml` - OpenCV face detector  
- `requirements.txt` - Python dependencies  

## How to run (local)
1. Clone repo  
2. Create virtual environment  
3. Install dependencies  
4. Run Streamlit or detection script
